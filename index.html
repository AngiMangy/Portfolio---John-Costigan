Welcome To My Portfolio

Hello! My name is John Costigan and this is my portfolio,

Projects

Quiz App

My quiz app is a simple ten question multiple choice quiz with a variety of topics ranging from Animal Facts to Sports. It keeps track of your score and gives you a final tally at the end. aswell as tracking how many times you have taken the quiz.
Skills used: JavaScript, HTML, CSS

Grocery List

The Grocery List is a simple form in which watever the user inputs gets added to a list below the form. It also has a simplistic sorting function to sort from "purchased", "unpurchased" and "all". A majority of the focus for this project was to show my understanding of JS interacting with HTML.
Skills used: JavaScript, HTML, CSS

Tic-Tac-Toe

A simple game of Tic-Tac-Toe that two players can play against eachother. Keeps a running tally of wins and ties. In the future I plan to add a single player mode where you can play against the computer. This project was also Focused on JS interaction with HTML elements.
Skills used: JavaScript, HTML, CSS
